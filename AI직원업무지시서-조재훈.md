# AI 직원 업무지시서 — 조재훈

> 작성일: 2026년 6월 4일

---

## 1. 신청자 맥락

**만들 AI 직원의 역할**
- 매주 네이버금융과 증권사 리포트에서 유량주 5개를 선정
- 각 종목별 선정 이유 작성
- 기업가치 관련 뉴스 요약
- 리포트 형태로 정리해 전달

**지금 겪는 문제**
- 주 2회 유량주 파악과 기업가치 뉴스를 직접 수동으로 확인 중
- 이 작업에 소요되는 시간이 투자 의사결정을 지연시킴

**이 자동화로 얻을 것**
- 투자 의사결정 시간 단축
- 향후 고객 판매 기회 창출 가능성

**사용자**: 조재훈 (유통·물류 업무 담당, ChatGPT 보유)

---

## 2. 오늘 만들 MVP (입력 1개 → 처리 1개 → 출력 1개)

### 입력값
조재훈님이 제공하는 정보:
- 네이버금융 URL 또는 증권사 리포트 링크 (또는 키워드)
- 분석 대상 기간 (예: "이번 주")

### 처리 흐름
1. **데이터 수집**: 제공된 URL/키워드에서 유량주 후보 정보 추출
2. **유량주 선정**: 5개 종목 선정
3. **분석**: 각 종목별 선정 이유 작성 + 기업가치 관련 뉴스 요약
4. **리포트 생성**: 정해진 템플릿에 맞춰 정리

### 출력값
**리포트 형식** (텍스트 또는 마크다운)
```
[유량주 분석 리포트]
분석 기간: YYYY-MM-DD
생성일: YYYY-MM-DD

【종목 1】
- 종목명 / 종목코드
- 선정 이유: [2-3줄]
- 관련 뉴스: [요약 1-2줄]
- 기업가치 평가: [간단한 평가]

【종목 2】
... (반복)

【종목 5】
...

주의: 본 리포트는 정보 제공 목적이며, 최종 투자 판단은 본인의 책임입니다.
```

## 3. 기술 구성 (MVP 범위)

**사용 도구**
- ChatGPT API (조재훈님 계정 또는 API 키)
- Python 또는 Node.js 스크립트
- 간단한 CLI 또는 웹 인터페이스 (입력창 1개 + 실행 버튼 + 결과 출력)

**데이터 소스**
- 네이버금융 (공개 정보)
- 증권사 리포트 (공개 링크 기준)
- ChatGPT를 통한 정보 종합 및 분석

## 6. 보안 및 주의사항

**API 키 관리**
- ChatGPT API 키는 `.env` 파일에 저장
- `.env` 파일은 `.gitignore`에 추가해 GitHub에 올리지 않기
- 로컬 테스트 시에만 사용

**개인정보 보호**
- 샘플 데이터로 테스트 (실제 투자 정보 X)
- 생성된 리포트에 "정보 제공 목적"임을 명시
- "최종 투자 판단은 본인의 책임"이라는 면책 조항 포함

**규제 리스크**
- 이 리포트는 "투자 조언"이 아닌 "정보 제공"으로 명확히 표기
- 종목 추천이 아닌 "분석 정보 제공"으로 표현

## 8. 성공 기준 (수업 당일)

**MVP 완성 기준**
1. 조재훈님이 URL 또는 키워드 1개를 입력
2. 실행 버튼 클릭

**검증 방법**
- 실제 네이버금융 데이터로 테스트 1회 실행
